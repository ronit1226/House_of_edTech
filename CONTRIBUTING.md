# Contributing

Thanks for helping improve ReadyAI.

## Development workflow

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Run the test suite and build:
   - npm test -- --run tests/unit/resume-interview.test.ts
   - npm run build
5. Open a pull request with a clear summary.

## Code style

- Keep code simple and readable.
- Prefer TypeScript and existing project patterns.
- Avoid introducing unused files or dead code.
